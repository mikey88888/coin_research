# log/

这里放自动化运行日志，给人看，不走机器优先的 JSON。

当前分两条轨道：

- `micro_optimization.log`：30 分钟微改进轨道
- `experiment_ranking.log`：2 小时实验/排名轨道

设计原则：

- **Append only**：只追加，不覆盖
- **人类可读**：优先清楚说明本轮想干什么、做了什么、结果如何
- **本地运行日志**：默认不进 git 历史，避免仓库长期被日志污染

推荐每轮至少记录：

- 时间
- run 类型 / run id（如果拿得到）
- 本轮目标
- 选择理由
- 实际执行的命令
- 改动文件
- 测试 / 回测结果
- 是否更新 leaderboard
- 是否跳过 / 阻塞
